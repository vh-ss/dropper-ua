# DropperUA — Buyer Tracker

Додаток обліку для Баєр-сервісу.

## Що це
- **Один офлайн HTML-файл**: `index.html` (HTML+CSS+JS в одному файлі, працює без сервера й без інтернету — просто відкрити у браузері).
- Дані/бекапи: `dropperua-backup-*.json` (gitignored), `dropperua-test-data.json` (демо).
- `localStorage`-ключ лишається `buyer_oblik_db` — НЕ перейменовувати (інакше злетять збережені дані).

## Межі проєкту
Працювати ТІЛЬКИ в цій папці (`TEST\DropperUA`). Не торкатися інших проєктів у `TEST`.

## Git
- Репозиторій: **vh-ss/Test_01**, гілка `main`.
- Після кожної зміни — самостійно `commit` + `push` (без додаткового запиту).
- Push працює лише з `http.sslBackend=schannel` (виставлено глобально).

## Робота
- Без збірки й залежностей — редагуєш `index.html` напряму.
- Перевірка: відкрити `index.html` у браузері.
