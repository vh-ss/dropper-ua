# Облік викупів / Buyer Tracker

Додаток для **Баєр-сервісу**: облік замовлень і викупів з іноземних сайтів (США/Європа)
на замовлення клієнтів з Instagram.

Це **один самодостатній файл** [`oblik.html`](oblik.html) — без сервера, без встановлення,
працює **офлайн**. Дані зберігаються локально на пристрої (`localStorage`).

## Можливості
- 📦 **Замовлення** — дата викупу, магазин, № замовлення, валюта (USD/EUR/PLN), статус,
  трек-номер, оплата клієнтом. Позиції: назва-артикул-розмір-кількість-закупка-ціна клієнту-посилання.
  Пошук по клієнту / магазину / № / треку.
- 👥 **Клієнти** — швидкий пошук, посилання на Instagram, усі замовлення клієнта, борг.
- 📉 **Ціни** — трекінг цін на обрані товари з історією, ↓/↑ і %, знижені зверху.
- 💰 **Дохід** — оборот і прибуток по валютах + загалом у базовій валюті за курсами.
- ⚙️ **Дані** — мова UA/EN, базова валюта + курси, експорт/імпорт `.json`, очистка.

## Як користуватись на телефоні
1. Завантаж `oblik.html` на телефон.
2. Відкрий у браузері (Safari / Chrome).
3. Меню → **«Додати на головний екран»**.
4. Періодично роби **Експорт** у вкладці «Дані» — це резервна копія.

## Локальний перегляд на ПК
```bash
python -m http.server 8765
# відкрий http://localhost:8765/oblik.html
```
