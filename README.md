# Dropper UA

**Облік замовлень і відправок для баєр-сервісу** — викуп товарів з іноземних сайтів на замовлення клієнтів з Instagram, з обліком доходів, боргів і доставки.

Один офлайн HTML-файл • без сервера • без інтернету • дані на пристрої

</div>

---

## ✨ Можливості

- 📦 **Замовлення** — магазин, №, дата викупу, статус, трек; товари з артикулом/розміром/фото; швидкі черги по статусах
- 👥 **Клієнти** — ПІБ, кілька телефонів, соцмережі (Instagram/TikTok/Threads/Facebook/Telegram), адреси (Нова Пошта/Укрпошта/Meest/кур'єр)
- 🏬 **Магазини** — окремі сутності з аналітикою
- 🚚 **На відправку** — товари, готові до пересилання, пакування в **посилки** з обов'язковим ТТН і **трекінгом** по оператору
- 💱 **Мульти-валюта** — закупка в USD/EUR/PLN, продаж у грн; курс фіксується на момент викупу; підтягування з **монобанку**
- 💰 **Дохід** — графік прибутку по місяцях, розрізи по магазинах/клієнтах, борги, **часткові оплати**
- 🌗 **Теми** — світла/темна/авто; 🇺🇦/🇬🇧 двомовний інтерфейс
- 💾 **Дані** — експорт/імпорт у файл, авто-знімок і відновлення

## 🚀 Як користуватись

1. Завантаж [`index.html`](index.html) на телефон.
2. Відкрий у браузері (Safari / Chrome).
3. Меню → **«Додати на головний екран»** — і користуйся як застосунком.
4. Час від часу роби **Експорт** у вкладці «Дані» (резервна копія).

Хочеш погратися з прикладом — імпортуй [`dropperua-test-data.json`](dropperua-test-data.json) (пів року тестових даних).

## 🛠 Технічно

Чистий HTML + CSS + JavaScript в одному файлі, без збірки й залежностей. Дані — у `localStorage` браузера.

```bash
# локальний перегляд на ПК
python -m http.server 8000
# відкрий http://localhost:8000/
```
